# Vichy Catalan

Vichy catalan info site, including how many you'd have to drink to actually get lithium poisoning.

## Deployment

This site uses **Cloudflare Workers Assets** for edge deployment with automatic CI/CD via GitHub Actions.

- **Production**: Push to `main` branch → deploys to `vichy-catalan-x7ppxm45.workers.dev`
- **Preview**: Create PR → automatic preview deployment with comment

## Project Structure

```
├── public/          # Static assets
│   └── index.html   # Main page (self-contained HTML with inline CSS/JS)
├── src/
```
